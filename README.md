# Weather DashBoard

Weather DashBoard is a weather widgets manager made with ReactJs, Redux & Webpack.

## Tech

For this project, I used :

- ReactJs
- Redux
- Webpack
- SASS
- ES6 with Babel
- Jest

## Installation

Clone the repo, then in the "weatherdashboard" folder:

```sh
$ cd weatherdashboard
$ npm install
$ npm start
```

Run tests:

```sh
$ npm test
```

Make a production build:

```sh
$ npm run build
```
